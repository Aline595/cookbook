# cookbook
O cookbook é uma aplicação de caderno de receitas.

## Funcionalidade

1. Adicionar receitas  
2. Listar receitas  
3. Deletar receita  

## Tecnologias
- Ruby  

## Como usar
### Pré-requisitos
  - É necessario possuir o Ruby pré instalaldo

 
### Passo a passo

1. Fazer o clone deste repositorio :
````
$ git clone https://github.com/Aline595/cookbook.git
````
3. Iniciar aplicação:
````
$ ruby cookbook.rb
````
